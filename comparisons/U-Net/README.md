# U-Net: Standard CNN Architecture

This folder contains training and inference scripts for U-Net, a classic convolutional neural network architecture widely used for medical image segmentation.

## Overview

U-Net is a fully convolutional network originally designed for biomedical image segmentation. It features an encoder-decoder architecture with skip connections that help preserve spatial information during upsampling.

## Prerequisites

Install required dependencies:

```bash
pip install torch torchvision
pip install monai
pip install segmentation-models-pytorch
```

## Architecture Features

- **Encoder-Decoder Structure**: Contracting path followed by an expanding path
- **Skip Connections**: Direct connection from encoder to decoder at each level
- **Efficient Parameter Usage**: Fewer parameters than typical CNN architectures

## Training

To train U-Net on your medical image dataset:

```bash
python train_unet.py \
    -i /path/to/input \
    -o /path/to/output \
    -b 32 \
    --max_epochs 500 \
    --num_workers 4
```

**Parameters:**
- `-i, --input`: Path to input dataset (npz format)
- `-o, --output`: Path to save trained model
- `-b, --batch_size`: Batch size (default: 32)
- `--max_epochs`: Maximum number of epochs (default: 500)
- `--num_workers`: Number of workers for data loading

## Inference

### 2D Images

```bash
python infer_unet_2D.py \
    -checkpoint /path/to/unet_best.pt \
    -data_root /path/to/input \
    -pred_save_dir /path/to/output \
    --save_overlay \
    -png_save_dir /path/to/overlay \
    -num_workers 2
```

### 3D Images

```bash
python infer_unet_3D.py \
    -checkpoint /path/to/unet_best.pt \
    -data_root /path/to/input \
    -pred_save_dir /path/to/output \
    --save_overlay \
    -png_save_dir /path/to/overlay \
    -num_workers 2
```

## Data Format

Input data should be in the `npz` format generated by MedSAM's preprocessing pipeline. For details, refer to [MedSAM preprocessing](https://github.com/bowang-lab/MedSAM#data-preprocessing).
