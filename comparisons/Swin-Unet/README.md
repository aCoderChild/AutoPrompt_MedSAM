# Swin-Unet: Transformer-based Medical Image Segmentation

This folder contains training and inference scripts for Swin-Unet, a hybrid transformer-CNN architecture using Swin Transformer blocks for medical image segmentation.

## Overview

Swin-Unet is a general-purpose vision transformer-based backbone with a U-shaped encoder-decoder architecture. It uses shifted window attention to achieve effective and efficient medical image segmentation with lower computational complexity compared to standard vision transformers.

## Prerequisites

Install required dependencies:

```bash
pip install torch torchvision
pip install timm
pip install einops
```

## Architecture Features

- **Swin Transformer Backbone**: Uses shifted window attention
- **U-shaped Architecture**: Encoder-decoder structure with skip connections
- **Multi-scale Features**: Multi-level feature extraction
- **Efficient Attention**: Shifted window reduces computational complexity
- **Patch-based Processing**: Hierarchical patch-based representation

## Training

To train Swin-Unet for medical image segmentation:

```bash
python train_swin_unet.py \
    -i /path/to/input \
    -o /path/to/output \
    -b 16 \
    --max_epochs 500 \
    --num_workers 4
```

**Parameters:**
- `-i, --input`: Path to input dataset (npz format)
- `-o, --output`: Path to save trained model
- `-b, --batch_size`: Batch size (default: 16)
- `--max_epochs`: Maximum number of epochs
- `--num_workers`: Number of workers for data loading

## Inference

### 2D Images

```bash
python infer_swin_unet_2D.py \
    -checkpoint /path/to/swin_unet_best.pt \
    -data_root /path/to/input \
    -pred_save_dir /path/to/output \
    --save_overlay \
    -png_save_dir /path/to/overlay
```

### 3D Images

```bash
python infer_swin_unet_3D.py \
    -checkpoint /path/to/swin_unet_best.pt \
    -data_root /path/to/input \
    -pred_save_dir /path/to/output \
    --save_overlay \
    -png_save_dir /path/to/overlay
```
