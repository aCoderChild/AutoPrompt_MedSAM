# Mask2Former: Transformer-based Hybrid Architecture

This folder contains training and inference scripts for Mask2Former, a transformer-based architecture that unifies instance, semantic, and panoptic segmentation.

## Overview

Mask2Former is a universal image segmentation model that combines transformer and convolutional features. It uses masked attention and learnable queries to achieve state-of-the-art performance in medical image segmentation tasks.

## Prerequisites

Install required dependencies:

```bash
pip install torch torchvision
pip install timm
pip install detectron2
pip install git+https://github.com/cocodataset/panopticapi.git
```

## Architecture Features

- **Transformer Backbone**: Vision transformer with multi-scale features
- **Masked Attention**: Reduces computational complexity
- **Unified Framework**: Handles multiple segmentation tasks
- **Learnable Queries**: Task-specific learnable embeddings

## Training

To train Mask2Former for medical image segmentation:

```bash
python train_mask2former.py \
    -i /path/to/input \
    -o /path/to/output \
    -b 16 \
    --max_epochs 500 \
    --num_workers 4
```

**Parameters:**
- `-i, --input`: Path to input dataset (npz format)
- `-o, --output`: Path to save trained model
- `-b, --batch_size`: Batch size (default: 16)
- `--max_epochs`: Maximum number of epochs
- `--num_workers`: Number of workers for data loading

## Inference

### 2D Images

```bash
python infer_mask2former_2D.py \
    -checkpoint /path/to/mask2former_best.pt \
    -data_root /path/to/input \
    -pred_save_dir /path/to/output \
    --save_overlay \
    -png_save_dir /path/to/overlay
```

### 3D Images

```bash
python infer_mask2former_3D.py \
    -checkpoint /path/to/mask2former_best.pt \
    -data_root /path/to/input \
    -pred_save_dir /path/to/output \
    --save_overlay \
    -png_save_dir /path/to/overlay
```

## Data Format

Expects input data in `npz` format generated by MedSAM's preprocessing pipeline.
